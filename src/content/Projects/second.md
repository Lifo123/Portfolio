# Demostración Completa de Markdown

Este documento muestra varios elementos de **Markdown** y cómo se pueden utilizar en un archivo. Markdown es un lenguaje de marcado ligero que permite formatear texto de manera sencilla y legible.

## Encabezados

Puedes crear diferentes niveles de encabezados usando `#`:

# Encabezado de Nivel 1
## Encabezado de Nivel 2
### Encabezado de Nivel 3
#### Encabezado de Nivel 4
##### Encabezado de Nivel 5
###### Encabezado de Nivel 6

## Párrafos

Los párrafos son simples bloques de texto. Solo necesitas dejar una línea en blanco entre ellos.

## Énfasis

Puedes enfatizar texto de las siguientes maneras:

- **Texto en negrita** se hace así.
- *Texto en cursiva* se hace así.
- **_Texto en negrita y cursiva_** se hace así.

## Listas

### Lista Desordenada

- Elemento 1
- Elemento 2
  - Subelemento 2.1
  - Subelemento 2.2
- Elemento 3

### Lista Ordenada

1. Primer elemento
2. Segundo elemento
3. Tercer elemento

## Imágenes

Puedes incluir imágenes de esta manera:

![Texto alternativo de la imagen](https://via.placeholder.com/300 "Título de la imagen")

## Tablas

Markdown también permite crear tablas de manera sencilla:

| Encabezado 1 | Encabezado 2 | Encabezado 3 |
|---------------|---------------|---------------|
| Fila 1 Col 1  | Fila 1 Col 2  | Fila 1 Col 3  |
| Fila 2 Col 1  | Fila 2 Col 2  | Fila 2 Col 3  |
| Fila 3 Col 1  | Fila 3 Col 2  | Fila 3 Col 3  |
